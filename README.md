# Auxita Code Challenege

## In case you don't have Vue.js installed
```
npm install -g @vue/cli
```
## Project setup
```
npm install
npm install sweetalert2
vue add unit-jest
npm install --save-dev @vue/test-utils
```
## Run the project
```
npm run serve
npm run build
```
## App running at:
```
Local: http://localhost:8080/
```
